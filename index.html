소개

소프트웨어과 전공동아리 중 제일 오래된 23년의 역사를 자랑하는 대표 웹 동아리입니다. 우수한 수상실적을 보유하고 있으며, 프론트엔드, 백엔드와 최신 웹개발 기술을 배울 수 있습니다!

커리큘럼

code

Lecture I

HTML, CSS, Javascript, GitHub을 학습하여 웹 기초를 다집니다.

description

Lecture II

Typescript, 모듈, Webpack, npm(모듈 배포)을 학습하여 프로젝트를 진행하기 위한 기반을 다집니다.

http

Lecture III

Node.js, Express, Socket.io, Vue를 학습하여 프로젝트를 진행해봅니다.

view_compact

Lecture IV, V

React, Context API, Svelte를 학습하여 SPA를 제작해봅니다.

QnA

Q. 아이왑에 지원하려면 실력이 얼마나 좋아야 하나요?

A. 완전 처음이여도 상관없습니다! 웹을 배우고자 하는 열정만 있으면 충분합니다!

Q. 포트폴리오를 반드시 내야 하나요?

A. 내든 안 내든 상관없으나 내면 좋습니다.

Q. 지원 자격이 어떻게 되나요?

A. 선린 1학년 소프트웨어과만 지원 가능합니다!
